Makefile
目录实现的功能

  • make compile 编译本application
  • make compile-deps 编译 application/deps/*
  • make compile-all
  • make install-deps copy applications in deps/ to /usr/lib/erlang/lib/目录下
  • make install copy 本applicatoin to /usr/lib/erlang/lib/目录下
